# C++ Learning

This repository documents my journey learning C++.

## Objectives

- Develop a solid foundation in programming.
- Learn object-oriented programming.
- Build the necessary foundation for embedded systems and engineering/robotics projects.

## Content

- **Environment Setup**: Compilers (GCC/Clang), IDEs (VS Code/Visual Studio), and Hello World.
- **Syntax & Structure**: Basic program anatomy, `#include`, `main()` function, and `std::cout`.
- **Variables & Data Types**: `int`, `float`, `double`, `char`, `bool`, `std::string`, and constants (`const`).
- **User Input**: Reading data from the console using `std::cin`.
- **Operators**: Arithmetic, assignment, comparison, and logical operators.
- **Control Flow (Conditionals)**: `if`, `else if`, `else`, and `switch-case` statements.
- **Loops (Iteration)**: `while`, `do-while`, and `for` loops.
- **Arrays**: Declaring, initializing, and iterating through fixed-size sequences.
